# Macro-Aware-Stress-Testing-System-MASTS-
An end-to-end MLOps pipeline for Credit Risk that integrates static borrower data with dynamic macroeconomic indicators (World Bank API). Using a Hybrid Gradient Boosting + SARIMAX architecture to adjust Risk Thresholds in real-time. Automations via GitHub Actions, version control with DVC, and model/drifts monitoring with Evidently AI.
